🟡 راهنمای نصب قالب HTML دارک‌نووا


✅ نصب قالب HTML دارک‌نووا روی لوکال (کامپیوتر شخصی)

  • ابتدا فایل قالب HTML در آدرس Theme/darknova.zip را دانلود کنید.
  • فایل darknova.zip را از حالت فشرده خارج (Extract) کنید.
  • در پوشه قالب، دنبال فایل index.html بگردید.
  • روی فایل index.html دوبار کلیک کنید. قالب در مرورگر شما باز خواهد شد.
  • اگر می‌خواهید قالب را ویرایش کنید، از نرم‌افزارهایی مثل VS Code یا Notepad++ استفاده کنید.

✅ نصب قالب HTML دارک‌نووا روی هاست (نمایش سایت در اینترنت)

  • ابتدا وارد پنل هاست خود شوید (cPanel یا DirectAdmin).
  • به قسمت File Manager بروید.
  • وارد پوشه‌ی public_html شوید.
  • اگر فایل‌های قبلی وجود دارد، آن‌ها را پاک کنید یا در یک پوشه دیگر منتقل کنید.
  • فایل قالب (darknova.zip) را در همان مسیر آپلود کنید.
  • روی فایل zip کلیک کرده و گزینه Extract را بزنید تا محتویات آن باز شود.
  • مطمئن شوید فایل index.html مستقیما داخل پوشه public_html قرار دارد.
  • حالا آدرس دامنه‌ات را در مرورگر بزن؛ قالب باید نمایش داده شود ✅
